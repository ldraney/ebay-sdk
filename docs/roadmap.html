Milestone 4: MCP Server

  • Separate repo: ebay-mcp-server

Progression per API Issue

  1. Read the OpenAPI spec (source of truth)
  2. Implement every endpoint as a method on the API class
  3. Write integration tests (real sandbox calls, real test objects)
  4. Verify all tests pass
  5. Close the GitHub issue

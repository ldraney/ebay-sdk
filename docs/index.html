ldraney-ebay-sdk

Python SDK wrapping all eBay REST APIs. The pure Python layer that auth, HTTP, and a future MCP server all build on.

$ pip install ldraney-ebay-sdk

What is this?

A thin, hand-written Python wrapper around every eBay REST API. You provide credentials via ldraney-ebay-oauth, and this SDK gives you typed methods for search, inventory, orders, finances, marketing, and more — no raw HTTP, no token juggling.

The end goal: this SDK becomes the foundation for an MCP server that lets AI agents interact with eBay programmatically.

Quick Start

from ebay_oauth import EbayOAuthClient
from ebay_sdk import EbayClient

oauth = EbayOAuthClient(client_id="...", client_secret="...")

with EbayClient(oauth) as ebay:
    results = ebay.buy_browse.search(q="vintage camera", limit=5)
    orders  = ebay.sell_fulfillment.get_orders(limit=5)
    cats    = ebay.commerce_taxonomy.get_category_suggestions("0", "camera")
